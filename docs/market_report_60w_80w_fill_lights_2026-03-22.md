# 60W-80W 便携型补光灯竞品对比报告

更新日期：2026-03-22

## 1. 报告目的

我写这份报告的目的很明确：如果后续把 `Photography_Fill_Light` 项目继续推进为便携型补光灯，我需要先看清它与市面主流 60W-80W 产品之间的差距，以及更合适的对标对象和价格区间。

## 2. 项目介绍

### 2.1 项目定位

我当前推进的是一个便携型补光灯方向的电子设计项目。项目的核心不是单独完成一块驱动板，而是围绕 `60W-80W` 功率段，逐步形成一套具备继续商品化潜力的补光灯方案。

从现有资料看，这个项目目前还处在样机与控制验证阶段，因此我对它的定义更偏向：

- 便携型补光灯方向的控制原型
- 以电源、恒流和调光为核心的技术样机
- 尚未完成整机工业设计和商品化指标闭环

### 2.2 目标方向

从当前讨论方向看，我把这个项目的目标理解为：

- 面向便携型补光灯赛道
- 功率落在 `60W-80W` 区间
- 强调稳定调光和连续光输出
- 后续需要与神牛、优篮子、智云等便携型产品做正面对比

从市场对比角度出发，我后续更关心的不是“能不能点亮”，而是它最终能否落到下面这些整机属性上：

- 是否足够便携
- 是否具备合适的供电方式
- 是否具备可接受的亮度和噪声表现
- 是否能进入合理价格区间

### 2.3 当前技术路线

结合现有文档和代码，我对本项目当前技术路线的概括是：

- 采用 `Boost + 线性恒流` 的总体架构
- 通过 `ADC + DMA` 进行采样与标定
- 通过 `PI` 控制完成母线调节
- 通过 `EC11` 完成交互与调光输入
- 已经具备软启动、过压保护和基本状态机逻辑

这条路线的特点是控制思路比较清晰，便于先把恒流、调光和动态余量控制做好；但如果继续朝便携型成品推进，后续还必须补齐结构、散热、供电、重量和噪声等整机问题。

### 2.4 当前完成情况

目前已经完成的部分主要包括：

- `ADC + DMA` 采样与标定
- `Boost + 线性恒流` 控制框架
- `PI` 调压、软启动、过压保护、状态机
- `EC11` 调光交互

目前还缺少、且无法直接与市售产品对比的数据主要包括：

- 1m 照度
- 整机重量
- 满功率风扇噪声
- 供电方式与续航能力
- 实际整机效率

基于这些现状，我目前认为本项目已经具备继续做市场对标分析的基础，但还没有进入“整机定型”的阶段。

## 3. 我关注的对比重点

在这类便携型补光灯上，我认为真正决定竞争力的不是单一功率，而是下面几项：

- 重量
- 供电方式
- 1m 照度
- 显指
- 风扇噪声
- 售价

市面主流 60W-80W 便携型产品，通常已经把这些点做到比较完整：

- 重量大致控制在 `250g-700g`
- 普遍支持 `PD`、`NP-F`、手柄电池或外接电池
- 有比较清晰的 1m 照度宣传口径
- 显指通常标到 `CRI 95+`
- 价格大致集中在 `579-1399 元`

而本项目当前最明显的差距是：

- 便携供电方案还没有定型
- 整机重量目标还没有锁定
- 照度、噪声、效率还没有实测数据
- 当前优势主要仍然停留在控制架构层面

## 4. 重点竞品对比

### 4.1 Godox ML60II Bi

`Godox ML60II Bi` 我把它视为便携专业型的基准产品之一。

公开可见参数大致为：

- 功率：`70W`
- 色温：`2800K-6500K`
- 显指：`CRI 96 / TLCI 97`
- 照度：`22500 lux @1m, 5600K, 带标罩`
- 重量：约 `526g`
- 供电：`AC`、`NP-F`、V 口扩展
- 价格：约 `898 元`

和本项目相比，我看到的差异主要是：

- 它已经完成整机形态
- 它的便携供电方案成熟
- 它的亮度口径清晰
- 它的品牌和附件生态完整

如果我后续把本项目定位为便携专业型，我认为至少要在下面几项尽量接近它：

- 重量
- 供电方式
- 1m 照度
- 风扇噪声

## 5. 我的直接结论

如果只看便携型路线，我当前更倾向于把本项目目标先收敛到下面这个范围：

- 功率：`60W-70W`
- 色温：`双色温`
- 供电：优先支持 `PD 100W`
- 重量：尽量进入 `700g` 以内
- 售价：优先落在 `599-899 元`

从竞品情况看，我当前对标优先级的判断是：

第一优先级：

- `Godox ML60II Bi`
- `Ulanzi EC65`

第二优先级：

- `ZHIYUN MOLUS X60`

第三优先级：

- `amaran COB 60x S`
- `SmallRig RC 60B`

我这样排序的原因也很直接：

- `ML60II Bi` 是便携专业型的主流基准
- `EC65` 是低价轻量路线的直接压力
- `X60` 代表小体积便携设计方向
- `amaran` 和 `SmallRig` 更适合做高位规格参考
